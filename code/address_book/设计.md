# 地址簿项目

## 功能需求

编写一款你自己的命令行_地址簿_程序，你可以用它浏览、添加、编辑、删除或搜索你的联系人，例如你的朋友、家人、同事，还有他们诸如邮件地址、电话号码等多种信息。这些详细信息必须被妥善储存以备稍后的检索。

## 系统设计

### 数据结构设计

联系人，包括姓名、性别、邮件地址、电话号码、生日、住址信息。类设计：[Contact.py](contact.py)

### 功能设计

#### 首页

程序启动列出程序功能，包括联系人分页列表、添加、编辑、删除、搜索、退出功能，界面如下：
```text
欢迎使用地址簿程序，当前地址簿总共有 xx 个联系人，请选择你需要的功能：
1. 联系人分页列表
2. 添加联系人
3. 编辑联系人
4. 删除联系人
5. 搜索联系人
6. 退出
```

联系人信息要做持久化，启动程序时从文件加载，退出时保存到文件

#### 联系人分页列表

每页只显示3个联系人，显示了当页的联系人信息以后，输入上一页、下一页功能选项，界面如下：
```text
用户id		姓名		性别		邮件地址		电话号码			生日				住址信息
1			a		男		<EMAIL>		12345678901		1990-01-01		402
2			b		女		<EMAIL>		12345678902		1990-01-02		403
3			c		男		<EMAIL>		12345678903		1990-01-03		404
当前页号：1，输入 b 返回上一页，输入 n 进入下一页 输入 q 退出列表
```

#### 添加联系人

添加联系人界面如下，添加成功后，为联系人创建唯一id（唯一id使用列表索引即可），并返回首页：
```text
请输入姓名（最多4个字符）：d
请输入性别（男/女）：男
请输入邮件地址（最多30个字符）：<EMAIL>
请输入电话号码（11个字符）：12345678904
请输入生日（yyyy-MM-dd）：1990-01-04
请输入住址信息（最多20个字符）：405
添加成功！
```

### 编辑联系人

编辑联系人界面如下，编辑成功后，将更新联系人信息，并返回首页：
```text
请输入需要修改的联系人id：1
请输入新的姓名（最多4个字符）：a
请输入新的性别（男/女）：男
请输入新的邮件地址（最多30个字符）：<EMAIL>
请输入新的电话号码（11个字符）：12345678901
请输入新的生日（yyyy-MM-dd）：1990-01-01
请输入新的住址信息（最多20个字符）：402
编辑成功！
```

### 删除联系人

删除联系人界面如下，删除成功后，将删除该联系人，并返回首页：
```text
请输入需要删除的联系人id：1
删除成功！
```

### 搜索联系人

搜索联系人界面如下，搜索成功后，将显示搜索结果，并返回首页：
```text
请输入需要搜索的信息：a
搜索结果如下：
用户id		姓名		性别		邮件地址		电话号码			生日				住址信息
1			a		男		<EMAIL>		12345678901		1990-01-01		402
搜索成功！
```
